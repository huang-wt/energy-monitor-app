## Deployment Manual
### Prerequisites
- Git
- Linux
- C++ 17 (and onwards)
- Qt
- CMake
### Installation
```shell
git clone https://github.com/huang-wt/energy-monitor-app.git
cd energy-monitor-app
```

### Build and Run
```shell
mkdir build && cd build
cmake ..
make
./energy_monitor
```
